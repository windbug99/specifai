# SpecClear - 3. 시장 분석

## 3.1 시장 규모

### 한국 시장

#### 전체 시장 규모
- **연간 IT 외주 시장 규모**: 약 5조원
- **평균 프로젝트 비용**: 2천만원
- **연간 프로젝트 수**: 약 25만 건

#### TAM-SAM-SOM 분석
```
TAM (Total Addressable Market)
└── 전체 IT 외주 시장: 5조원

SAM (Serviceable Available Market)
└── 중소 규모 외주 프로젝트: 2조원 (40%)

SOM (Serviceable Obtainable Market)
└── 초기 목표 시장 (0.1% 침투): 250건 → 연 매출 1.25억원
```

#### 세그먼트별 시장 규모

| 세그먼트 | 시장 규모 | 프로젝트 수 | 평균 단가 |
|----------|----------|------------|-----------|
| **스타트업** | 1조원 | 5만 건 | 2천만원 |
| **중소기업** | 2조원 | 10만 건 | 2천만원 |
| **프리랜서** | 1조원 | 10만 건 | 1천만원 |
| **대기업** | 1조원 | 1천 건 | 10억원 |

## 3.3 시장 트렌드

### SpecClear가 타이밍이 완벽한 이유

#### ✅ Trend 1: AI 코딩 폭발
```
2023년: Copilot 출시
2024년: Cursor 인기 폭발
2025년: AI 코딩 주류화 ← 우리는 여기

영향:
• 명확한 명세 = AI 활용 극대화
• 애매한 명세 = AI 헤맴
→ 명세서 품질이 개발 속도를 결정
```

#### ✅ Trend 2: 프리랜서 경제
```
변화:
• 정규직 → 프리랜서 전환 가속
• 단기 계약 증가
• 신뢰 부족 문제 심화

영향:
• 명세서 필요성 증가
• 계약 문서 중요성 상승
→ SpecClear 수요 증가
```

#### ✅ Trend 3: 1인 개발자 증가
```
현상:
• AI로 혼자서도 제품 개발 가능
• 기술 창업 증가
• 부트스트랩 스타트업 증가

니즈:
• 명확한 개발 계획 필요
• AI와의 효율적 협업 필요
→ 명세서가 개발 로드맵 역할
```

#### ✅ Trend 4: 원격 근무
```
변화:
• 대면 소통 감소
• 문서 중심 업무 증가
• 비동기 협업 일반화

영향:
• 문서 품질 = 프로젝트 성공
→ 명세서가 핵심 소통 도구
```

#### ✅ Trend 5: 법적 분쟁 증가
```
현황:
• 디지털 계약 증가
• 외주 분쟁 다발
• 법적 보호 니즈 증가

해결책:
• 명확한 명세서 = 분쟁 방지
• 디지털 서명 = 법적 효력
→ SpecClear 가치 상승
```

### 시장 진입 타이밍 분석

#### Technology Adoption Lifecycle
```
혁신자 → 초기수용자 → 초기다수 → 후기다수 → 지각수용자
        ↑
    우리는 여기
    (초기수용자)

이유:
• AI 코딩 도구 사용자 = 초기수용자
• 외주 분쟁 경험자 = 페인포인트 인식
• 솔루션 갈망 = 지불 의향 높음
```

## 3.4 시장 진입 전략

### Phase별 시장 공략

#### Phase 1: 니치 시장 (0-6개월)
- **타겟**: AI 코딩 도구 사용하는 1인 개발자
- **채널**: ultravisionlab, 개발 커뮤니티
- **목표**: 100명 얼리어답터 확보

#### Phase 2: 확장 (6-12개월)
- **타겟**: 스타트업 창업자, 프리랜서 개발자
- **채널**: 위시켓/이랜서 파트너십
- **목표**: 500명 활성 사용자

#### Phase 3: 주류화 (12-24개월)
- **타겟**: 중소기업, 에이전시
- **채널**: B2B 영업, 파트너십
- **목표**: 2,000명 사용자, 시장 리더
