# Specifai - 6. 비즈니스 모델

## 6.1 수익 모델 (소규모 프로젝트 시장 중심)

### 수익원 구성

#### 주요 수익원 (우선순위)
```
1. 프로젝트당 과금 (70%) ← 1인 개발자, 프리랜서 중심
   └── 명세서 작성 비용 (소규모 프로젝트 중심)

2. 구독 모델 (25%) ← 향후 확장
   └── Pro/Enterprise 플랜 (팀 협업 기능)

3. 부가 서비스 (5%) ← 향후
   └── 법률 검토, 컨설팅 (기업 고객)
```

### 프로젝트당 과금 모델 (소규모 프로젝트 중심)

#### 1차 타겟: 1인 개발자 & 프리랜서용 가격
| 프로젝트 유형 | 가격 | 포함 사항 | 타겟 |
|--------------|------|----------|------|
| **Personal** | 5만원 | • 명세서 1개<br>• 수정 2회<br>• PDF/Markdown 출력 | 1인 개발자<br>(개인 프로젝트) |
| **Freelancer** | 20만원 | • 명세서 1개<br>• 수정 무제한<br>• AI 프롬프트 생성<br>• 고객 공유 링크 | 프리랜서<br>(소규모 프로젝트) |
| **Business** | 50만원 | • Freelancer 모든 기능<br>• 양측 리뷰 & 협의<br>• 변경 요청 관리<br>• 90일 지원 | 소규모 팀<br>(여러 프로젝트) |

#### 2차 타겟: 기업 고객용 가격 (향후 추가)
| 프로젝트 규모 | 가격 | 포함 사항 | 타겟 |
|--------------|------|----------|------|
| **Enterprise** | 100만원+ | • 무제한 프로젝트<br>• 팀 협업 기능<br>• 커스텀 패키지<br>• 전담 매니저 | 규모있는 기업<br>(여러 소규모 팀) |

#### 가격 정당성 (소규모 프로젝트 시장)
```
1인 개발자:
• 기존: AI 도구 사용 (재시도 5회, 2주 소요, 저품질)
• Specifai: 명확한 명세 (재시도 1회, 2-3일 소요, 고품질)
• 절감: 시간 7배 단축 = 시간 × 비용으로 환산 시 ROI 매우 높음

프리랜서:
• 기존: 분쟁 위험 = 평균 미수금 50만원
• Specifai: 명세서 20만원으로 분쟁 예방
• 절감: 분쟁 비용 vs 명세서 비용 = 2.5배 이상 이득

소규모 팀:
• 기존: 명세서 부실로 인한 예산 초과 (평균 16-20%)
• Specifai: 명확한 범위로 예산 통제
• 절감: 5,000만원 프로젝트 × 16% = 800만원 절감
```


## 6.5 단위 경제 (Unit Economics)

### CAC vs LTV 분석

#### 고객 획득 비용 (CAC)
```
마케팅 비용: 20만원
영업 비용: 10만원
온보딩 비용: 5만원

총 CAC: 35만원
```

#### 고객 생애 가치 (LTV)
```
평균 구매 횟수: 3회
평균 구매액: 30만원
총 매출: 90만원
기여이익률: 65%

LTV = 90만원 × 0.65 = 58.5만원

LTV/CAC = 58.5 / 35 = 1.67
```

### 개선 목표
```
현재: LTV/CAC = 1.67 (마진 있음)
6개월: LTV/CAC = 2.5 (건전함)
12개월: LTV/CAC = 3.0 (우수함)
```

### 개선 방안

#### CAC 감소
- 콘텐츠 마케팅 (기술 블로그, 개발자 커뮤니티)
- 추천 프로그램
- 파트너십 활용

#### LTV 증가
- 구독 모델 전환
- 업셀/크로스셀
- 고객 만족도 향상
