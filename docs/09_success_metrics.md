## 9.7 성공 정의

### 단계별 성공 기준

#### Phase 1: Validation (0-3개월)
```
✅ Success Criteria:
□ 제품-시장 적합성 확인
□ 유료 고객 10명 이상
□ NPS 40 이상
□ 재구매 의향 50% 이상

🚫 Failure Criteria:
□ 유료 전환 5% 미만
□ NPS 0 이하
□ 100% 이탈
```

#### Phase 2: Traction (3-12개월)
```
✅ Success Criteria:
□ MRR 1,000만원 달성
□ 고객 100명 확보
□ 데이터 해자 구축
□ 첫 투자 유치

🚫 Failure Criteria:
□ 성장률 0%
□ CAC > LTV
□ 팀 구성 실패
```

#### Phase 3: Scale (1-3년)
```
✅ Success Criteria:
□ 시장 리더 포지션
□ ARR 10억원
□ 해외 진출
□ Exit 기회

🚫 Failure Criteria:
□ 경쟁사에 추월
□ 성장 정체
□ 자금 소진
```
