# SpecClear - 1. 개요

**서비스명:** SpecClear (스펙클리어)  
**부제:** 분쟁 없는 외주 개발을 위한 명확한 요구사항 명세서 작성 플랫폼  
**버전:** 1.1  
**작성일:** 2025년 1월

---

## 1.1 서비스 설명

SpecClear는 IT 외주 개발 시 가장 큰 문제인 **발주자와 개발자 간의 요구사항 불일치로 인한 분쟁**을 해결하는 플랫폼입니다. AI 기반 대화형 인터페이스를 통해 비개발자도 쉽게 명확하고 법적 효력이 있는 요구사항 명세서를 작성할 수 있도록 지원합니다.

### 주요 특징
- AI 기반 대화형 인터페이스
- 비개발자 친화적 질문 방식
- 법적 효력이 있는 계약 문서 수준의 명세서
- AI 코딩 도구(Cursor, Claude)용 프롬프트 자동 생성

---

## 1.2 핵심 가치 제안

### 정량적 가치
- **분쟁 방지**: 명확한 범위 설정으로 외주 분쟁 80% 감소
- **시간 절감**: AI 도움으로 명세서 작성 시간 70% 단축 (2주 → 2시간)
- **비용 절감**: 명세서 부실로 인한 예산 초과 방지 (평균 30% 절감)
- **AI 구현 최적화**: 명확한 명세로 AI 코딩 도구 활용 시 재시도 5회 → 1회
- **법적 보호**: 디지털 서명과 변경 추적으로 법적 분쟁 시 증거 확보

### 정성적 가치
- 발주자와 개발자 간의 신뢰 구축
- 프로젝트 범위에 대한 명확한 합의
- 개발 과정의 투명성 확보
- 추가 요구사항 발생 시 체계적 관리

## 1.3 타겟 사용자

### 주요 타겟 그룹

| 타겟 | 설명 | 핵심 니즈 |
|------|------|-----------|
| **외주 발주자** | 스타트업 창업자, 비개발 기획자, 중소기업 대표 등 IT 개발을 외주로 진행하려는 개인 및 기업 | • 명확한 요구사항 전달<br>• 예산 초과 방지<br>• 분쟁 예방 |
| **프리랜서 개발자** | 고객과 직접 계약하는 개인 개발자 및 소규모 개발 에이전시로 범위 분쟁 예방 필요 | • 명확한 작업 범위<br>• 추가 요청 관리<br>• 계약 보호 |
| **1인 개발자** | AI 코딩 도구(Cursor, Claude 등)를 활용하는 개인 개발자로 명확한 명세 작성으로 AI 구현 효율 극대화 | • AI 활용 극대화<br>• 구현 효율성<br>• 재작업 최소화 |

### 타겟 사용자별 페인포인트

#### 외주 발주자
- "개발자와 대화가 안 통해요"
- "예상보다 비용이 너무 많이 나왔어요"
- "내가 원했던 것과 다른 결과물이 나왔어요"

#### 프리랜서 개발자
- "고객이 계속 기능을 추가해달라고 해요"
- "처음 약속과 달라진 요구사항 때문에 힘들어요"
- "완료 후에도 계속 수정 요청이 들어와요"

#### 1인 개발자 (AI 활용)
- "AI에게 명확한 지시를 내리기 어려워요"
- "여러 번 재시도해도 원하는 결과가 안 나와요"
- "프로젝트 전체 구조를 AI가 이해하지 못해요"

## 1.4 제품 포지셔닝

### 시장 내 위치
```
범용 문서 도구 ←→ SpecClear ←→ 복잡한 기업용 솔루션
     ChatGPT                      aqua cloud

단순/저렴 ←→ 외주 특화/적정가 ←→ 복잡/고가
```

### 핵심 차별화
1. **외주 계약 전용**: 일반 문서가 아닌 외주 계약에 특화
2. **AI + 도메인 지식**: 단순 AI가 아닌 외주 도메인 전문성 결합
3. **양방향 관점**: 발주자와 개발자 모두를 고려한 설계
4. **데이터 기반 개선**: 실제 프로젝트 데이터로 지속적 학습

---

## 1.5 비전과 미션

### 비전
"모든 IT 외주 프로젝트가 명확한 합의와 신뢰 속에서 성공하는 세상"

### 미션
"AI와 도메인 전문성을 결합하여 발주자와 개발자 간의 소통 격차를 해소하고, 명확한 요구사항 명세서를 통해 외주 프로젝트의 성공률을 높인다"

### 핵심 가치
- **명확성(Clarity)**: 애매함 없는 명확한 소통
- **신뢰(Trust)**: 발주자와 개발자 간의 신뢰 구축
- **효율성(Efficiency)**: 시간과 비용의 효율적 활용
- **공정성(Fairness)**: 양측 모두에게 공정한 계약
