# Specifai MVP

AI 기반 요구사항 명세서 작성 플랫폼의 MVP (Minimum Viable Product)

## 🎯 개요

Specifai는 사용자가 객관식 질문에 답변하는 것만으로 명확한 요구사항 명세서를 생성할 수 있는 플랫폼입니다. 이 MVP는 핵심 기능을 검증하기 위해 구현되었습니다.

### 주요 기능

1. **초기 입력**: 사용자가 자유롭게 프로젝트 아이디어를 입력
2. **AI 분석**: 입력된 내용을 분석하여 프로젝트 유형과 복잡도 파악
3. **객관식 질문** (15개): 한국 시장에 최적화된 질문으로 요구사항 구체화
4. **실시간 명세서 생성**: 답변할 때마다 실시간으로 명세서 생성
5. **검토 및 수정**: 완성된 명세서 확인 및 수정
6. **문서 출력**:
   - Markdown 명세서 (계약서 수준)
   - AI 프롬프트 (Cursor, Claude 등에서 바로 사용 가능)

## 🚀 시작하기

### 필수 요구사항

- Node.js 18+
- npm 또는 yarn

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 을 열어 확인하세요.

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm start
```

## 📁 프로젝트 구조

```
app/
├── src/
│   ├── app/              # Next.js 앱 라우터 페이지
│   │   ├── page.tsx      # 홈페이지
│   │   ├── create/       # 1단계: 초기 입력
│   │   ├── analyze/      # 2단계: AI 분석
│   │   ├── questions/    # 3단계: 객관식 질문
│   │   ├── review/       # 4단계: 검토
│   │   └── result/       # 5단계: 최종 결과
│   ├── components/       # 재사용 가능한 컴포넌트
│   ├── lib/              # 유틸리티 함수
│   │   ├── context.tsx   # React Context (상태 관리)
│   │   ├── questions.ts  # 질문 데이터 (15개)
│   │   ├── analyzer.ts   # 프로젝트 분석 로직
│   │   └── generator.ts  # 명세서 생성 로직
│   └── types/            # TypeScript 타입 정의
├── public/               # 정적 파일
└── package.json          # 의존성
```

## 🔧 기술 스택

- **Frontend**: Next.js 14 (App Router) + React 18
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **State Management**: React Context
- **Deployment**: Vercel (권장)

## 📝 MVP 제약사항

이 MVP는 다음과 같은 제약사항을 가지고 있습니다:

### 고정값 (컨텍스트)
- 발주자: 프로젝트 담당 PM
- 개발자: 프리랜서 5년차
- 국가: 한국
- 언어: 한국어

### 고정 질문
- 이커머스(쇼핑몰) 프로젝트용 15개 질문
- 한국 시장 최적화 (카카오, 네이버 우선)

### 제한된 기능
- PDF 출력 없음 (Markdown만 제공)
- 실제 AI API 연동 없음 (키워드 기반 분석)
- 사용자 계정 없음
- 프로젝트 저장 기능 없음

## 🎨 UX 플로우

```
홈페이지
   ↓
1. 초기 입력 (자유 서술형)
   ↓
2. AI 분석 결과 확인
   ↓
3. 객관식 질문 (15개)
   - 단일 선택 / 다중 선택
   - 실시간 명세서 미리보기
   ↓
4. 검토 및 수정
   - 선택한 기능 확인
   - 예상 개발 기간 및 비용 확인
   ↓
5. 최종 문서 출력
   - Markdown 명세서 다운로드
   - AI 프롬프트 다운로드
```

## 🧪 테스트 시나리오

MVP를 테스트하려면 다음 시나리오를 따라해보세요:

1. 홈페이지에서 "프로젝트 시작하기" 클릭
2. 초기 입력 화면에서 다음과 같이 입력:
   ```
   소규모 쇼핑몰을 만들고 싶습니다.
   의류를 판매하고, 회원 관리와 결제 기능이 필요합니다.
   ```
3. AI 분석 결과를 확인하고 "네, 맞습니다" 클릭
4. 15개 질문에 답변 (각 질문은 프로젝트에 따라 선택)
5. 검토 화면에서 선택한 기능과 예상 비용 확인
6. 최종 결과 화면에서 명세서와 AI 프롬프트 다운로드
